HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines the CSS styles for a website.
  • The base element defines the base url for a webpage.
  • The link element connects and external resource to the HTML document.
  • The meta element defines metadata(character set, description, keywords, author, viewport)

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Periods preceding elements designates them as class selectors
  • CSS files must be linked to the HTML framework or they cannot interact
  • CSS resources

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git command cheat sheet

JavaScript

  • JavaScript Basics
  • Variables are containers that store values. Start with 'let' and the variables name
  • An operator is a mathematical symbol that produces a result based on two values
  • Conditionals are used to test if an expression is true or not(if/else)
  • Functions can be used to call a body of code that executes when you call the function name in your code
  • Event handlers check for activity in the browser and run code in response
  • JavaScript data types and structures
